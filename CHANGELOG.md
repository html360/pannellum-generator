## 2.5.0 (2026-04-02)
- Initial release of the standalone Pannellum generator.
- Cross-platform binaries for Windows, Linux, and macOS.
- Automatic binary downloader in postinstall script.
- Node.js API to retrieve the executable path.
- CLI support via html360-gen command.
- GitHub Actions pipeline for automated builds and releases.
- Core script compatibility with Pillow 10.0+ and NumPy.
- Bundled nona engine from Hugin for cube projection.
